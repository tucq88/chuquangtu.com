# chuquangtu.com

Personal landing page for Tu Chu — plain static HTML/CSS/JS, no build step.

## Develop

Serve the directory locally:

```
python3 -m http.server        # http://localhost:8000
```

Or just open `index.html` in a browser.

## Deploy

Static host (GitHub Pages / Netlify / Cloudflare Pages) — serve the repo root; `index.html` is the entry point.
